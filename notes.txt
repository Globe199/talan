For things like sending the <ENTER> key, or keybinds to bring windows up they must
be sent using PostMessage using WM_KEYDOWN. The game requires KEYUP/KEYDOWN for <ENTER>,
but otherwise seems ok with WM_CHAR for text in a dialog box.

Bring up a crafting window bound to 'n':
	    PostMessage(tmp, WM_KEYDOWN, 'N', 0);

Use Careful Synthesis II four times:
		char str[] = "/ac \"Careful Synthesis II\"";
		for (int i = 0; i < 4; i++) {
			PostMessage(tmp, WM_KEYDOWN, VK_RETURN, 0);
			PostMessage(tmp, WM_KEYUP, VK_RETURN, 0);
			Sleep(250);
			for (char& c : str) {
				PostMessage(tmp, WM_CHAR, c, 0);
			}
			Sleep(250);
			PostMessage(tmp, WM_KEYDOWN, VK_RETURN, 0);
			PostMessage(tmp, WM_KEYUP, VK_RETURN, 0);
			Sleep(3000);
		}
